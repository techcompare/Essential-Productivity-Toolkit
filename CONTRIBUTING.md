# Contributing to Essential Productivity Toolkit

Thank you for your interest in contributing to the Essential Productivity Toolkit! This guide will help you contribute effectively.

## How to Contribute

### Adding a New Tool or Resource

1. **Fork the repository** to your GitHub account
2. **Clone your fork** locally
3. **Create a new branch** for your contribution:
   ```bash
   git checkout -b add-new-tool
   ```
4. **Add your tool/resource** to the appropriate section in README.md
5. **Commit your changes** with a clear message:
   ```bash
   git commit -m "Add [Tool Name] to [Section]"
   ```
6. **Push to your fork**:
   ```bash
   git push origin add-new-tool
   ```
7. **Submit a pull request** from your fork to the main repository

### Contribution Guidelines

#### Tool Selection Criteria

Tools added to this collection should meet the following criteria:

- **Quality**: The tool should be well-designed and reliable
- **Relevance**: Must directly contribute to productivity, focus, or organization
- **Availability**: Should be publicly available (free or paid)
- **Maintenance**: Actively maintained with recent updates
- **Personal Experience**: Ideally, you should have used the tool personally

#### Formatting Requirements

Each tool entry should follow this format:

```markdown
- **[Tool Name](https://tool-url.com/)** - Brief one-line description
```

Example:
```markdown
- **[Todoist](https://todoist.com/)** - Powerful task management with natural language
```

#### Section Placement

Place your contribution in the most appropriate section:

- **Time Management Tools** - Timers, trackers, calendars, scheduling
- **Focus & Concentration** - Distraction blockers, ambient sound, meditation
- **Task Management & Organization** - To-do lists, project management, kanban
- **Note-Taking & Knowledge Management** - Digital notes, knowledge bases, bookmarks
- **Communication & Collaboration** - Team chat, video conferencing, email
- **Automation & Productivity Systems** - Automation tools, password managers, file storage
- **Wellness & Work-Life Balance** - Health apps, sleep tracking, digital wellbeing
- **Development & Learning Resources** - Courses, methodologies, books

If your tool doesn't fit any existing category, suggest a new one in your pull request description.

#### Alphabetical Order

Maintain alphabetical order within each subsection to make tools easier to find.

### Suggesting New Sections

If you believe a new category would benefit the toolkit:

1. Open an issue describing the proposed section
2. Explain why it's necessary and what it would contain
3. Provide 3-5 example tools that would fit in this section
4. Wait for community feedback before creating a pull request

### Reporting Issues

Found a broken link or outdated information?

1. Open an issue with a clear title
2. Describe the problem
3. If reporting a broken link, suggest an alternative if possible

### Code of Conduct

#### Our Standards

- Be respectful and inclusive
- Welcome newcomers and help them learn
- Focus on what is best for the community
- Show empathy towards others

#### Unacceptable Behavior

- Harassment or discriminatory language
- Trolling or insulting comments
- Spam or self-promotion without value
- Publishing others' private information

### Questions?

If you have questions about contributing, open an issue with the label "question" and we'll be happy to help!

## Recognition

Contributors will be recognized in future versions of this project. Thank you for helping make productivity tools more accessible to everyone!
