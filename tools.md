This section contains practical tools and applications that support everyday productivity.

Only tools that are reliable, widely used, and genuinely useful for getting work done are included.

## All-in-One Workspaces

- [Notion](https://www.notion.so) - Versatile workspace combining notes, databases, and project management.
- [Evernote](https://evernote.com) - Note-taking app with powerful search and organization features.
- [Obsidian](https://obsidian.md) - Knowledge base on local Markdown files (free for personal use).
- [ClickUp](https://clickup.com) - All-in-one productivity platform with docs, tasks, and goals.
- [Coda](https://coda.io) - All-in-one doc that brings words, data, and teams together.
- [Airtable](https://airtable.com) - Spreadsheet-database hybrid for organizing anything.

## Automation Tools

- [Zapier](https://zapier.com) - Automation platform connecting thousands of apps without code.
- [IFTTT](https://ifttt.com) - Simple automation tool for connecting apps and devices.
- [n8n](https://n8n.io) - Open-source workflow automation (self-hosted free).
- [Make (Integromat)](https://www.make.com) - Visual automation platform for complex workflows.
- [Automatisch](https://automatisch.io) - Open-source Zapier alternative.
- [Power Automate](https://powerautomate.microsoft.com) - Microsoft's automation platform.
