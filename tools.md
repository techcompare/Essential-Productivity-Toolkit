This section contains practical tools and applications that support everyday productivity.

Only tools that are reliable, widely used, and genuinely useful for getting work done are included.

## Resources

- [Notion](https://www.notion.so) - Versatile workspace combining notes, databases, and project management.
- [Evernote](https://evernote.com) - Note-taking app with powerful search and organization features.
- [Zapier](https://zapier.com) - Automation platform connecting thousands of apps without code.
- [IFTTT](https://ifttt.com) - Simple automation tool for connecting apps and devices.
- [Alfred](https://www.alfredapp.com) - Productivity app for macOS with workflows and snippets.
- [Raycast](https://www.raycast.com) - Extendable launcher for macOS with built-in productivity tools.
- [1Password](https://1password.com) - Password manager for secure credential storage and sharing.
- [Pocket](https://getpocket.com) - Save articles and videos to read or watch later.
- [Grammarly](https://www.grammarly.com) - Writing assistant for grammar, spelling, and clarity.
- [TextExpander](https://textexpander.com) - Snippet expansion tool to automate repetitive typing.
