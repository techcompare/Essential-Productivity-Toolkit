## Contributing

We welcome contributions to the Essential Productivity Toolkit! If you have a tool, resource, or system that has significantly improved your productivity, please consider adding it.

### How to Contribute

1. Fork this repository
2. Add your tool/resource to the appropriate section
3. Ensure your addition includes:
   - Tool name with link
   - Brief description (one line)
   - Proper formatting consistent with existing entries
4. Submit a pull request with a clear description

### Guidelines

- Only include tools you have personally used or researched thoroughly
- Keep descriptions concise and objective
- Ensure tools are actively maintained and available
- Place tools in the most relevant category
- Maintain alphabetical order within each subsection
